UNDERGRADUATE
DEGREE IN
CYBER SECURITY

A degree in Cyber Security will give the understanding of computer networks and computer systems, along with the ability to both break into and fix the holes that are in them. This degree will also give the knowledge required to both protect and exfiltrate data from systems around the world.

Some fun classes that you will take include:


 - Cloud Architecture and Security

 - Data Communication & Networking

 - Network Forensics

 - Network Security
